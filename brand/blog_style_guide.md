# Arthur Blog Style Guide

## Structure
- **Meta title**: 50-60 characters. Include primary keyword. No clickbait.
- **Meta description**: 150-160 characters. Summarize the post's value prop for the reader.
- **Intro paragraph**: Hook the reader in 2-3 sentences. State the problem or question the post answers.
- **H2 sections**: 3-5 major sections. Each H2 should be scannable and descriptive.
- **Conclusion**: 1-2 paragraphs. Summarize key takeaway + CTA.

## Length Guidelines
- Standard blog post: 1,200-1,800 words
- Deep-dive / anchor content: 1,800-2,500 words
- Column / opinion piece: 800-1,200 words

## SEO Conventions
- Primary keyword in: meta title, H1, first 100 words, at least one H2, meta description.
- Use related keywords naturally throughout — don't stuff.
- Internal links: 3-5 per post to other Arthur content (blog posts, columns, product pages).
- External links: Only to authoritative sources (regulatory bodies, analyst firms, peer-reviewed research). Open in new tab.

## Content Rules
- Lead with the reader's problem, not Arthur's product.
- Use concrete examples and scenarios relevant to the target audience's industry.
- Data and citations: always link to primary sources. No unsourced statistics.
- Avoid: first person ("I think"), hedging language ("perhaps," "maybe"), filler transitions.
- OK to use: "we" when referring to Arthur as a company in appropriate context.

## Formatting
- Use bullet lists for 3+ parallel items.
- Use numbered lists only for sequential steps.
- Bold key phrases for scannability — but not entire sentences.
- Code blocks or technical notation where relevant.
- One CTA at the end. Optionally one mid-post CTA for long-form content.
